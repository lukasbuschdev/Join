Déclaration de confidentialité

1. Protection des données en un coup d'œil

Informations générales

Les informations suivantes fournissent un aperçu simple de ce qui se passe avec vos données personnelles lorsque vous visitez ce site Web. Les données personnelles sont toutes les données grâce auxquelles vous pouvez être personnellement identifié. Pour plus d'informations sur le sujet de la protection des données, veuillez vous référer à notre politique de protection des données répertoriée ci-dessous.

Collecte de données sur ce site Web

Qui est responsable de la collecte de données sur ce site Web ?

Le traitement des données sur ce site Web est effectué par l'exploitant du site. Vous pouvez trouver ses coordonnées dans la section "Informations sur l'organisme responsable" de cette politique de confidentialité.

Comment collectons-nous vos données ?

D'une part, vos données seront collectées lorsque vous nous les communiquez. Cela peut être par exemple des données que vous saisissez dans un formulaire de contact.

D'autres données sont collectées automatiquement ou avec votre consentement par nos systèmes informatiques lorsque vous visitez le site Web. Il s'agit principalement de données techniques (par exemple, navigateur Internet, système d'exploitation ou heure de consultation de la page). Ces données sont collectées automatiquement dès que vous entrez sur ce site Web.

Pourquoi utilisons-nous vos données ?

Certaines données sont collectées pour garantir la fourniture sans erreur du site Web. D'autres données peuvent être utilisées pour analyser votre comportement d'utilisateur.

Quels sont vos droits concernant vos données ?

Vous avez le droit de recevoir à tout moment des informations gratuites sur l'origine, la réception et l'objectif de vos données personnelles stockées. Vous avez également le droit de demander la correction ou la suppression de ces données. Si vous avez donné votre consentement au traitement des données, vous pouvez révoquer ce consentement à tout moment pour l'avenir. De plus, dans certaines circonstances, vous avez le droit de demander la limitation du traitement de vos données personnelles. De plus, vous avez le droit de porter plainte auprès de l'autorité de contrôle compétente.

Vous pouvez nous contacter à tout moment pour poser des questions sur la protection des données et d'autres questions connexes.

Pour cela, ainsi que pour d'autres questions concernant les données personnelles, vous pouvez nous contacter à tout moment.

Droit à la limitation du traitement

Vous avez le droit de demander la limitation du traitement de vos données personnelles. À cette fin, vous pouvez nous contacter à tout moment. Le droit à la limitation du traitement existe dans les cas suivants :

  • Si vous contestez l'exactitude de vos données personnelles stockées par nous, nous avons généralement besoin de temps pour vérifier cela. Pendant la durée de la vérification, vous avez le droit de demander la limitation du traitement de vos données personnelles.
  • Si le traitement de vos données personnelles a été effectué/de déroule de manière incorrecte, vous pouvez demander la limitation du traitement des données au lieu de la limitation.
  • Si nous n'avons plus besoin de vos données personnelles, mais que vous en avez besoin pour exercer, défendre ou faire valoir une revendication légale, vous avez le droit de demander la limitation du traitement de vos données personnelles au lieu de la limitation.
  • Si vous avez fait objection en vertu de l'art. 21, paragraphe 1, du RGPD, un équilibre doit être établi entre vos intérêts et les nôtres. Tant que l'on n'a pas encore déterminé quels intérêts prédominent, vous avez le droit de demander la limitation du traitement de vos données personnelles.

Si vous avez limité le traitement de vos données personnelles, ces données, à l'exception de leur stockage, ne peuvent être traitées qu'avec votre consentement ou aux fins de faire valoir, d'exercer ou de défendre des droits légaux ou de protéger les droits d'une autre personne physique ou morale ou pour des raisons d'intérêt public important de l'Union européenne ou d'un État membre. Si la limitation du traitement a été limitée conformément aux conditions ci-dessus, vous serez informé par nous avant que la limitation ne soit levée.

4. Collecte de données sur ce site Web

Cookies

Nos pages Internet utilisent des cookies. Les cookies sont de petits fichiers texte et ne causent aucun dommage à votre appareil. Ils sont stockés soit temporairement pour la durée d'une session (cookies de session) ou de manière permanente (cookies permanents) sur votre appareil. Les cookies de session sont automatiquement supprimés après la fin de votre visite. Les cookies permanents restent stockés sur votre appareil jusqu'à ce que vous les supprimiez vous-même ou que votre navigateur Web les supprime automatiquement.

Dans de nombreux cas, les cookies remplissent des fonctions techniques essentielles. Par exemple, certains cookies sont nécessaires pour que le site Web conserve son aspect et sa convivialité, en permettant par exemple de stocker vos préférences linguistiques. Nous utilisons également des cookies sur notre site Web qui permettent d'analyser le comportement de navigation de nos visiteurs (par exemple, les pages visitées, le temps passé sur le site Web). Cela nous permet d'améliorer constamment la qualité de notre site Web et de fournir une meilleure expérience utilisateur.

Les cookies sont stockés sur votre appareil et transmettent des informations à notre serveur. Cela signifie que nous, en tant qu'exploitant du site Web, obtenons directement des informations sur votre utilisation de notre site Web. En outre, certains cookies (en particulier les cookies de suivi) peuvent également être utilisés par des tiers. Les cookies de ce type nous permettent, par exemple, de mesurer le succès de nos publicités en ligne ou d'analyser votre comportement de navigation à des fins de marketing.

Vous pouvez configurer votre navigateur pour être informé de l'installation de cookies et décider individuellement de les accepter ou d'exclure l'acceptation de cookies dans certains cas ou en général. Vous pouvez également activer la suppression automatique des cookies lorsque vous fermez votre navigateur. Si vous désactivez les cookies, la fonctionnalité de ce site Web peut être limitée.

Fichiers journaux du serveur

Le fournisseur des pages collecte et stocke automatiquement des informations dans des fichiers journaux de serveur, que votre navigateur nous transmet automatiquement. Ce sont :

  • Type et version du navigateur
  • Système d'exploitation utilisé
  • URL de référence
  • Nom d'hôte de l'ordinateur accédant
  • Heure de la demande du serveur
  • Adresse IP

Une fusion de ces données avec d'autres sources de données n'est pas effectuée.

La collecte de données est basée sur l'art. 6, paragraphe 1, lit. f DSGVO. L'exploitant du site Web a un intérêt légitime à garantir la sécurité et la stabilité de son site Web. Pour ce faire, les fichiers journaux du serveur doivent être collectés.
