Ajuda

Bem-vindo à página de ajuda do Join, seu guia para usar nossa ferramenta de gerenciamento de projetos Kanban. Aqui oferecemos uma visão geral do que é o Join, como ele pode ser útil para você e como usá-lo.

O que é o Join?

O Join é uma ferramenta de gerenciamento de projetos baseada em Kanban, projetada e desenvolvida por um grupo de estudantes dedicados como parte de seu bootcamp de desenvolvimento web na Developer Akademie.

O Kanban, um termo japonês que significa "quadro", é um método extremamente eficaz para visualizar o trabalho, limitar o trabalho em andamento e maximizar a eficiência (ou o fluxo). O Join utiliza os princípios do Kanban para ajudar os usuários a gerenciar suas tarefas e projetos em uma interface intuitiva e visual.

É importante observar que o Join é concebido como um exercício na área de educação e não é destinado a aplicativos de negócios extensos. Embora nos esforcemos para garantir a melhor experiência do usuário possível, não podemos garantir disponibilidade constante, confiabilidade, precisão ou outros aspectos de qualidade relacionados ao Join.

Como usá-lo

Aqui está um guia passo a passo para usar o Join:

  • Explorando o quadro

    Quando você se inscrever no Join, terá a opção de criar um novo quadro. Este quadro representa seu projeto e inclui quatro listas padrão:

    • "A fazer"
    • "Em andamento"
    • "Aguardando feedback"
    • "Concluído"

    Você pode adicionar novos cartões ao quadro pressionando o botão "Nova tarefa". Também é possível pesquisar conteúdo específico dos cartões no quadro atualmente selecionado.

  • Adicionar contatos

    No Join, você pode adicionar contatos para colaborar em seus projetos. Vá para a seção "Contatos", clique no botão "Novo contato" e insira o nome de usuário. Uma vez selecionado o contato e pressionado o botão "Adicionar contato", será enviada uma solicitação que o destinatário deverá confirmar. Depois disso, você pode atribuir tarefas a esse e a outros contatos já adicionados ao quadro, desde que seus contatos tenham sido adicionados anteriormente ao quadro relevante.

  • Adicionar cartões

    Na página "Nova tarefa", você pode adicionar cartões. Os cartões representam tarefas individuais. Clique no botão "Adicionar tarefa" para criar um novo cartão. Comece selecionando um quadro e preencha os detalhes da tarefa no cartão, como:

    • Título
    • Descrição
    • Categoria
    • Membros
    • Data de vencimento
    • Prioridade

    (Subtarefas são opcionais.)

  • Mover cartões

    Quando uma tarefa passa de uma fase para outra, você pode mostrar isso no quadro arrastando e soltando o cartão de uma lista para outra.

  • Editar cartões

    Ao clicar em um cartão, ele se abrirá e você poderá editar todos os detalhes da tarefa usando o botão "Editar".

  • Excluir cartões

    Quando uma tarefa estiver concluída, você poderá movê-la para a lista "Concluído" ou excluí-la. A exclusão de um cartão o removerá permanentemente do quadro. Tenha cuidado ao excluir cartões, pois essa ação não pode ser desfeita.

  • Cartão do usuário

    Se você clicar no botão no canto superior direito com sua foto de perfil, abrirá um cartão com as seguintes opções:

    • Notificações

      Aqui você verá notificações relacionadas a um quadro ao qual foi adicionado, além de solicitações de contato.

    • Editar conta

      Você pode editar sua foto de perfil, nome de usuário, email, senha e cor de perfil aqui.

    • Idioma

      Você pode definir o idioma do usuário usando os botões disponíveis.

    • Sair

      Você pode sair usando este botão.
