Ajuda

Bem-vindo à página de ajuda do Join, seu guia para usar nossa ferramenta de gerenciamento de projetos Kanban. Aqui, oferecemos uma visão geral do que é o Join, como ele pode ser útil para você e como usá-lo.

O que é o Join?

O Join é uma ferramenta de gerenciamento de projetos baseada em Kanban, projetada e desenvolvida por um grupo de estudantes dedicados como parte de seu bootcamp de desenvolvimento web na Developer Akademie.

O Kanban, um termo japonês que significa "quadro", é um método altamente eficaz para visualizar o trabalho, limitar o trabalho em andamento e maximizar a eficiência (ou fluxo). O Join utiliza os princípios do Kanban para ajudar os usuários a gerenciar suas tarefas e projetos em uma interface intuitiva e visual.

É importante notar que o Join foi concebido como um exercício no campo da educação e não é destinado a aplicações comerciais extensivas. Embora nos esforcemos para garantir a melhor experiência do usuário possível, não podemos garantir disponibilidade constante, confiabilidade, precisão ou outros aspectos de qualidade em relação ao Join.

Como usar

Aqui está um guia passo a passo sobre como usar o Join:

  • Explorando o Quadro

    Ao fazer login no Join, você encontrará um quadro padrão. Este quadro representa seu projeto e contém quatro listas padrão: "A Fazer", "Em Andamento", "Aguardando Feedback" e "Concluído".

  • Criando Contatos

    No Join, você pode adicionar contatos para colaborar em seus projetos. Vá para a seção "Contatos", clique em "Novo Contato" e insira as informações necessárias. Depois de adicionados, você pode atribuir tarefas a esses contatos e eles podem interagir com as tarefas no quadro.

  • Adicionando Cartões

    Após adicionar seus contatos, você pode adicionar cartões. Cartões representam tarefas individuais. Clique no botão "+" sob a lista correspondente para criar um novo cartão. Preencha os detalhes da tarefa no cartão, como nome da tarefa, descrição, data de vencimento, responsável, etc.

  • Movendo Cartões

    Quando uma tarefa avança de uma fase para a próxima, você pode mostrar isso no quadro movendo o cartão de uma lista para outra.

  • Excluindo Cartões

    Quando uma tarefa é concluída, você pode movê-la para a lista "Concluído" ou excluí-la. A exclusão de um cartão o remove permanentemente do quadro. Por favor, tenha cuidado ao excluir cartões, pois este processo não pode ser desfeito. Lembre-se de que o uso eficaz do Join requer atualizações regulares de você e sua equipe para garantir que o quadro reflita o estado atual do seu projeto. Tem mais perguntas sobre o Join?
