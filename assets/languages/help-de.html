Hilfe

Willkommen auf der Hilfe-Seite für Join, Ihre Anleitung zur Verwendung unseres Kanban-Projektmanagement-Tools. Hier bieten wir einen Überblick darüber, was Join ist, wie es Ihnen nutzen kann und wie Sie es verwenden können.

Was ist Join?

Join ist ein auf Kanban basierendes Projektmanagement-Tool, das von einer Gruppe engagierter Studenten im Rahmen ihres Webentwicklungsbootcamps an der Developer Akademie entworfen und entwickelt wurde.

Kanban, ein japanischer Begriff, der "Plakatwand" bedeutet, ist eine äußerst effektive Methode zur Visualisierung von Arbeit, zur Begrenzung von Arbeiten im Gange und zur Maximierung der Effizienz (oder des Flusses). Join nutzt die Prinzipien von Kanban, um Benutzern bei der Verwaltung ihrer Aufgaben und Projekte in einer intuitiven, visuellen Oberfläche zu helfen.

Es ist wichtig zu beachten, dass Join als Übung im Bildungsbereich konzipiert ist und nicht für umfangreiche geschäftliche Anwendungen vorgesehen ist. Obwohl wir uns bemühen, das bestmögliche Benutzererlebnis sicherzustellen, können wir keine gleichbleibende Verfügbarkeit, Zuverlässigkeit, Genauigkeit oder andere Qualitätsaspekte in Bezug auf Join garantieren.

Wie man es verwendet

Hier ist eine Schritt-für-Schritt-Anleitung zur Verwendung von Join:

  • Das Board erkunden

    Wenn Sie sich bei Join anmelden, haben Sie die Möglichkeit ein neues Board zu erstellen. Dieses Board repräsentiert Ihr Projekt und enthält vier Standardlisten:

    • "Zu erledigen"
    • "In Arbeit"
    • "Feedback abwarten"
    • "Erledigt"

    Sie können der Tafel neue Karten hinzufügen, indem Sie auf den Button "Neue Aufgabe" drücken. Ebenfalls möglich ist die Suche nach gezielten Inhalten der Karten der momentan ausgewählten Tafel.

  • Kontakte hinzufügen

    In Join können Sie Kontakte hinzufügen, um an Ihren Projekten zusammenzuarbeiten. Gehen Sie zum Abschnitt "Kontakte", klicken Sie auf den Button "Neuer Kontakt" und geben Sie den Benutzernamen ein. Sobald sie den Kontakt ausgewählt und auf den Button "Kontakt hinzufügen" gedrückt haben, wird eine Anfrage gesendet welche der Empfänger erst bestätigen muss. Danach können Sie diesem und anderen bereits vorhandenen Kontakten Aufgaben zuweisen, insofern Ihre Kontakte vorher zur jeweiligen Tafel hinzugefügt wurde.

  • Karten hinzufügen

    Auf der Seite "Neue Aufgabe" können Sie Karten hinzufügen. Karten repräsentieren einzelne Aufgaben. Klicken Sie auf den Button "Aufgabe hinzufügen", um eine neue Karte zu erstellen. Wählen Sie zuerst eine Tafel und füllen Sie danach die Aufgabendetails in der Karte aus, wie:

    • Titel
    • Beschreibung
    • Kategorie
    • Mitglieder
    • Fälligkeitsdatum
    • Priorität

    (Unteraufgaben sind optional.)

  • Karten verschieben

    Wenn die Aufgabe von einer Phase zur nächsten übergeht, können Sie dies auf dem Board anzeigen, indem Sie die Karte von einer Liste in eine andere ziehen und ablegen.

  • Karten bearbeiten

    Wenn Sie auf eine Karte klicken, öffnet sich diese und man kann mittels des Buttons "Bearbeiten" alle Aufgabendetails bearbeiten.

  • Karten löschen

    Wenn eine Aufgabe abgeschlossen ist, können Sie sie entweder in die "Erledigt"-Liste verschieben oder löschen. Das Löschen einer Karte entfernt sie dauerhaft vom Board. Bitte seien Sie vorsichtig beim Löschen von Karten, da dieser Vorgang nicht rückgängig gemacht werden kann.

  • Benutzer Karte

    Wenn Sie auf den oben rechts, in der Ecke befindlichen Button mit Ihrem Profilbild drücken, öffnet sich eine Karte mit folgenden Auswahlmöglichkeiten:

    • Benachrichtigungen

      Hier sehen Sie zu einer Tafel hinzugefügt wurden, sowie Kontaktanfragen.

    • Konto bearbeiten

      Hier können Sie Ihr Profilbild, sowie Benutzernamen, E-Mail, Passwort und Profilfarbe bearbeiten.

    • Sprache

      Hier können Sie mittels der Buttons Ihre Benutzersprache festlegen

    • Abmelden

      Mittels dieses Buttons können Sie sich Ausloggen.

Denken Sie daran, dass die effektive Nutzung von Join regelmäßige Updates von Ihnen und Ihrem Team erfordert, um sicherzustellen, dass das Board den aktuellen Stand Ihres Projekts widerspiegelt.
