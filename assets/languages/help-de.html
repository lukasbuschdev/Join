Hilfe

Willkommen auf der Hilfe-Seite für Join, Ihre Anleitung zur Verwendung unseres Kanban-Projektmanagement-Tools. Hier bieten wir einen Überblick darüber, was Join ist, wie es Ihnen nutzen kann und wie Sie es verwenden können.

Was ist Join?

Join ist ein auf Kanban basierendes Projektmanagement-Tool, das von einer Gruppe engagierter Studenten im Rahmen ihres Webentwicklungsbootcamps an der Developer Akademie entworfen und entwickelt wurde.

Kanban, ein japanischer Begriff, der "Plakatwand" bedeutet, ist eine äußerst effektive Methode zur Visualisierung von Arbeit, zur Begrenzung von Arbeiten im Gange und zur Maximierung der Effizienz (oder des Flusses). Join nutzt die Prinzipien von Kanban, um Benutzern bei der Verwaltung ihrer Aufgaben und Projekte in einer intuitiven, visuellen Oberfläche zu helfen.

Es ist wichtig zu beachten, dass Join als Übung im Bildungsbereich konzipiert ist und nicht für umfangreiche geschäftliche Anwendungen vorgesehen ist. Obwohl wir uns bemühen, das bestmögliche Benutzererlebnis sicherzustellen, können wir keine gleichbleibende Verfügbarkeit, Zuverlässigkeit, Genauigkeit oder andere Qualitätsaspekte in Bezug auf Join garantieren.

Wie man es verwendet

Hier ist eine Schritt-für-Schritt-Anleitung zur Verwendung von Join:

  • Das Board erkunden

    Wenn Sie sich bei Join anmelden, finden Sie ein Standard-Board. Dieses Board repräsentiert Ihr Projekt und enthält vier Standardlisten: "Zu erledigen", "In Arbeit", "Feedback abwarten" und "Erledigt".

  • Kontakte erstellen

    In Join können Sie Kontakte hinzufügen, um an Ihren Projekten zusammenzuarbeiten. Gehen Sie zum Abschnitt "Kontakte", klicken Sie auf "Neuer Kontakt" und geben Sie die erforderlichen Informationen ein. Sobald sie hinzugefügt sind, können diesen Kontakten Aufgaben zugewiesen und sie können mit den Aufgaben auf dem Board interagieren.

  • Karten hinzufügen

    Nachdem Sie Ihre Kontakte hinzugefügt haben, können Sie Karten hinzufügen. Karten repräsentieren einzelne Aufgaben. Klicken Sie auf die "+"-Schaltfläche unter der entsprechenden Liste, um eine neue Karte zu erstellen. Füllen Sie die Aufgabendetails in der Karte aus, wie Aufgabenname, Beschreibung, Fälligkeitsdatum, Zuständige usw.

  • Karten verschieben

    Wenn die Aufgabe von einer Phase zur nächsten übergeht, können Sie dies auf dem Board anzeigen, indem Sie die Karte von einer Liste in eine andere ziehen und ablegen.

  • Karten löschen

    Wenn eine Aufgabe abgeschlossen ist, können Sie sie entweder in die "Erledigt"-Liste verschieben oder löschen. Das Löschen einer Karte entfernt sie dauerhaft vom Board. Bitte seien Sie vorsichtig beim Löschen von Karten, da dieser Vorgang nicht rückgängig gemacht werden kann. Denken Sie daran, dass die effektive Nutzung von Join regelmäßige Updates von Ihnen und Ihrem Team erfordert, um sicherzustellen, dass das Board den aktuellen Stand Ihres Projekts widerspiegelt.
