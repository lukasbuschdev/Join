Help

Welcome to the help page for Join, your guide to using our Kanban project management tool. Here we provide an overview of what Join is, how it can benefit you, and how to use it.

What is Join?

Join is a Kanban-based project management tool designed and developed by a group of dedicated students as part of their web development bootcamp at the Developer Academy.

Kanban, a Japanese term meaning "billboard," is an extremely effective method for visualizing work, limiting work in progress, and maximizing efficiency (or flow). Join uses the principles of Kanban to help users manage their tasks and projects in an intuitive, visual interface.

It's important to note that Join is designed as an educational exercise and is not intended for extensive business applications. While we strive to ensure the best possible user experience, we cannot guarantee consistent availability, reliability, accuracy, or other quality aspects related to Join.

How to Use It

Here is a step-by-step guide to using Join:

  • Registration & Verification

    When registering a new account with Join, after entering a username, email address, and password, and confirming the privacy policy, a verification email is sent. To confirm the account, you can click on the link in the email or enter and submit the verification code manually or via copy & paste on the verification page.

  • Create an Account

    It is possible to upload (and delete) a profile picture and choose a profile color.

  • Login

    Login with optional saving of login data.

  • Reset Password

    Resetting the password works like verification via email.

  • Automatic Logout

    Automatic logout after 5 minutes of inactivity.

  • Notifications & Websocket Server

    When sending a contact request, collaborator invitation, or assigning a task to a collaborator, the recipient receives a live notification if they are online. A Websocket server has been implemented for this.

  • Account Settings

    Clicking on your profile at the top right opens the account settings. Here you can view notifications, accept or decline contact requests and board invitations, reset your password, or delete your account. You can also log out here.

  • Languages

    The language is loaded according to the default language set in the browser. Language settings are also available during login and in the account settings.

  • Create a Board

    There is not only no default board, but you must also first create one or be invited to another (for newly created accounts). You can set a name, define categories, and invite collaborators.

  • Edit a Board

    After creation, the board can be edited (only by the creator), collaborators can be added via invitation (but not removed), and categories can be added or deleted.

  • Delete a Board

    Entire boards can also be deleted (with confirmation).

  • Select a Board

    To select the active board, you must first click on "Overview"/"Summary" and then, if necessary, click on "Select Board". Then select the desired board from the list.

  • Create Tasks

    Individual tasks (including subtasks) can be created and assigned to collaborators of the corresponding board.

  • Edit/Delete Tasks

    Tasks can be edited and deleted.

  • Move Tasks

    Tasks can be moved between the four types to do, in progress, awaiting feedback, and done via drag & drop in the board view.

  • Contact & Collaborator Request

    When adding a contact or a collaborator, a request is sent that must be confirmed by the recipient before being saved in the contacts/board.

  • Add Contacts

    To add a new contact, at least part of a name must be entered in the search field before search results are displayed. When adding the contact, a contact request is sent to them, which must be confirmed or declined.
