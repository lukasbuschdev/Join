Help

Welcome to the Help page for Join, your guide to using our Kanban project management tool. Here, we provide an overview of what Join is, how it can benefit you, and how to use it.

What is Join?

Join is a Kanban-based project management tool designed and developed by a group of dedicated students as part of their web development bootcamp at the Developer Academy.

Kanban, a Japanese term meaning "billboard," is an extremely effective method for visualizing work, limiting work in progress, and maximizing efficiency (or flow). Join utilizes the principles of Kanban to assist users in managing their tasks and projects in an intuitive, visual interface.

It's important to note that Join is designed for educational purposes and is not intended for extensive business applications. While we strive to provide the best user experience possible, we cannot guarantee consistent availability, reliability, accuracy, or other quality aspects of Join.

How to Use It

Here is a step-by-step guide on how to use Join:

  • Explore the Board

    When you log in to Join, you have the option to create a new board. This board represents your project and includes four default lists:

    • "To Do"
    • "In Progress"
    • "Waiting for Feedback"
    • "Done"

    You can add new cards to the board by clicking the "New Task" button. It's also possible to search for specific content within the cards of the currently selected board.

  • Add Contacts

    In Join, you can add contacts to collaborate on your projects. Go to the "Contacts" section, click the "New Contact" button, and enter the username. Once you've selected the contact and clicked the "Add Contact" button, a request will be sent, which the recipient must confirm. Afterward, you can assign tasks to this contact and others if they have been added to the respective board previously.

  • Add Cards

    You can add cards on the "New Task" page. Cards represent individual tasks. Click the "Add Task" button to create a new card. First, select a board, and then fill out the task details in the card, such as:

    • Title
    • Description
    • Category
    • Members
    • Due Date
    • Priority

    (Subtasks are optional.)

  • Move Cards

    When a task transitions from one phase to another, you can display it on the board by dragging and dropping the card from one list to another.

  • Edit Cards

    Clicking on a card opens it, and you can edit all the task details using the "Edit" button.

  • Delete Cards

    When a task is completed, you can either move it to the "Done" list or delete it. Deleting a card permanently removes it from the board. Please be cautious when deleting cards, as this action cannot be undone.

  • User Card

    When you click on the button located in the upper right corner with your profile picture, a card with the following options opens:

    • Notifications

      Here, you can see notifications related to boards you've been added to, as well as contact requests.

    • Edit Account

      Here, you can edit your profile picture, username, email, password, and profile color.

    • Language

      Here, you can set your user language using the buttons.

    • Log Out

      Use this button to log out.
